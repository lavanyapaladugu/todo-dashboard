# Todo Dashboard

A full-stack Todo application built using React, Node.js, and Express with file-based storage.

## Features
- Add, delete, and mark tasks as completed
- Filter tasks (All, Completed, Pending)
- Real-time search functionality
- Persistent storage using JSON file
- Unique ID-based task management
- REST API integration

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Storage: JSON file (fs module)

## Run Locally

### Backend
cd server
node server.cjs

### Frontend
npm install
npm run dev

##  Project Highlights
- Full-stack application with CRUD operations
- Dynamic UI updates using React Hooks
- Backend integration with REST APIs
